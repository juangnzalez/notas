# Uso del plugin ViM-TeX

Fuente https://github.com/lervag/vimtex

**Instalación**

```
   Plug 'lervag/vimtex'
```

## Inicio rápido

- Iniciar compilación con `\ll`
- Abrir *pdf* generado con `\lv`
- Ver errores con `\le`
- Limpiar archivos auxiliarer con `\lc`, con `\lC` eliminamos tambien el *pdf*
